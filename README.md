# Cat App

A simple web application for posting and browsing cat listings.

## Features

* User registration
* Add a new cat
* View cat catalog
* Contact the cat owner

## Database Setup

Create a database named `cat_app` and configure your MySQL credentials.

## Run the app

python app.py

Open: http://127.0.0.1:5000
